# CodeHealth — Product Hunt Launch Kit

## Tagline (60 chars max)
Know your code's true health in 60 seconds

## Description
Submit any GitHub repo. Get a professional AI-powered audit covering security vulnerabilities, dependency risks, code quality, test coverage, and compliance — delivered as a shareable report or downloadable PDF.

No setup. No config files. No DevOps team required.

**6 audit dimensions in under 60 seconds:**
- 🛡️ Security Analysis — OWASP-aligned vulnerability detection
- 📦 Dependency Audit — CVEs, outdated packages, license risks
- 📊 Code Quality — Health score, complexity, anti-patterns
- 🧪 Test Coverage — Tested vs untested file mapping
- 📋 Compliance — GDPR, HIPAA, PCI-DSS readiness
- 📄 PDF Reports — Professional, shareable, white-label ready

**Free to start.** 3 scans/month. Pro at $29/mo for teams.
